Colour is a shader package let you achieve nice gradient color effect without lighting and texture.
It includes 3 shaders:
  => "Colour/Surface" is the surface shader let you do gradient effect.
  => "Colour/Sky1" is a top-bottom gradient skybox shader.
  => "Colour/Sky2" is a top-middle-bottom gradient skybox shader.
The demo scene demonstrates all features, please refer to it as example usage.
Open it and click play button, Hold left mouse button and move can rotate the camera. Hold W,S,A,D,Q,E key will move the camera.
